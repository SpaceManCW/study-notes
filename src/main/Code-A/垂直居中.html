• width/height/top/bottom都为0，margin为auto，父元素relative。子元素绝对定位
  • margin负值,top和left都设置为50%,然后给margin的top和left设置负值
  • 利用Flex布局的justify-content和align-items属性
  • 父容器设置table-cell,vertical-align: middle,text-align: center;子元素设置display: inline-block

1、如果同时设置top、bottom、left、right，margin设置值中没有auto，那么位置top值优先于bottom值，left值优先于right值

2、如果不设置位置，只设置margin:auto;那么布局中相当于margin:0;。

其对立定位方向属性同时有具体定位数值的时候，流体特性就发生了。子元素大小适应于父元素的padding box宽度。
